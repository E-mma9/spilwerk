# Contracten — wanneer welke gebruiken

Vier templates, elk voor een andere situatie. **Niet alle klusjes hebben een contract nodig** — voor een €55-bezoek is een korte WhatsApp-bevestiging genoeg.

> ⚠️ Disclaimer: starter-templates voor B2C / klein-zakelijk gebruik. Voor projecten >€2.500 of contracten met gemeentes/zorginstellingen: laat ze één keer doorlezen door een MKB-jurist (~€150-300 eenmalig).

## Beslisboom — welke gebruik je wanneer?

```
Eénmalig klusje (<€200, particulier)
    → Opdrachtbevestiging mini (WhatsApp/SMS)

Eénmalig groter werk (>€200, vaste prijs, zelfstandige/klein bedrijf)
    → Dienstverleningsovereenkomst (1 A4)

Maandelijks terugkerend werk
    → Onderhoudsovereenkomst (2 A4)

Je verwerkt persoonsgegevens namens de klant
(M365, gedeelde mappen, klantenlijsten, mailbox)
    → Verwerkersovereenkomst — naast bovenstaand
```

## Welke ondertekening werkt?

Vier opties, in oplopende formaliteit:

1. **WhatsApp-akkoord** — "akkoord, gaan we morgen doen" — geldig voor klusjes <€200, voor particulier
2. **E-mail "akkoord"** — verstuurd vanaf hun zakelijk e-mailadres, met de PDF van de overeenkomst in de thread — geldig voor klein-zakelijk tot ~€1.500
3. **Getekende PDF terug** — voor projecten >€1.500
4. **DocuSign / Yousign / SignRequest** (€10/mnd) — als je 5+ contracten/maand stuurt, dan worth it. Eerder niet.

Voor jou nu: 90% gaat via optie 1-2. Niets ingewikkelds nodig.

## Aansprakelijkheid blijft begrensd via je AV

Alle bovenstaande contracten verwijzen naar je `algemene-voorwaarden.md`. Daar staat (artikel 8): aansprakelijkheid begrensd op factuurbedrag, geen indirecte schade, etc. **Stuur de AV altijd mee als bijlage** bij elk getekend contract.
